
# Teletyper

 platform dedicated to show who's the better typer


## FYI 

The folders are labeled after the show: Avatar the Last Airbender.
Folder names:
- Frontend = roku
- Backend = aang

## Technologies used
### Frontend
- Vue 3
- Nuxt
- TailwindCSS
- DaisyUI

### Backend
- Spring
- MongoDB

## Installation

Frontend

```bash
  cd roku
  npm install
  npm run dev
```
    
Backend

```bash
  cd aang
  gradle build --refresh-dependencies
```
    
Based Text Editor:
```bash
  https://nvchad.com/docs/quickstart/install
```
